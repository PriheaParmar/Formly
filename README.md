# Formly

**Type once. Chill later.**

Formly remembers repeated form answers and helps fill them next time, because retyping is exhausting.

## Install

1. Unzip the file.
2. Open Chrome and go to `chrome://extensions`.
3. Turn on **Developer mode**.
4. Click **Load unpacked**.
5. Select the `formly-extension` folder.

## How to use

1. Open any website with a form.
2. Type your answer in a field.
3. Click outside the field.
4. Formly will ask if you want to save it.
5. Click **Save**.

Next time you open the same form:

1. Click the empty field.
2. Formly will find your saved answer.
3. Click **Fill**.

## Manage saved answers

1. Click the Formly extension icon.
2. Click **Manage**.
3. Use the manage page to:
   - search saved answers
   - expand or collapse sites
   - reveal answers
   - copy answers
   - delete answers

## Reveal PIN

The first time you reveal or copy a saved answer, Formly will ask you to set a PIN.

Use this PIN whenever you want to view or copy saved answers.

## Notes

Formly does not save passwords, hidden fields, file inputs, checkboxes, or radio buttons.

Formly does not submit forms automatically.
