# 💜 Formly

### Type once. Chill later.

Formly remembers repeated form answers and helps fill them next time,  
because retyping the same thing again is honestly exhausting.

<br>

## ✦ Install

1. Unzip the project.
2. Open Chrome.
3. Go to:

```txt
chrome://extensions
````

4. Turn on **Developer mode**.
5. Click **Load unpacked**.
6. Select the folder:

```txt
formly-extension
```

---

## ✦ Save an answer

1. Open any website with a form.
2. Type your answer.
3. Click outside the field.
4. Formly will ask if you want to save it.
5. Click **Save**.

That’s it. Formly remembers it for next time.

---

## ✦ Fill next time

1. Open the same form again.
2. Click the empty field.
3. Formly will find your saved answer.
4. Click **Fill**.

No retyping. No drama.

---

## ✦ Manage answers

Click the **Formly** extension icon, then click **Manage**.

From there you can:

* search saved answers
* expand or collapse websites
* reveal saved answers
* copy answers
* delete answers

---

## ✦ Reveal PIN

The first time you reveal or copy an answer, Formly asks you to set a PIN.

After that, use your Formly PIN whenever you want to view or copy saved answers.

---

## ✦ Notes

Formly does **not** save:

* passwords
* hidden fields
* file inputs
* checkboxes
* radio buttons

Formly also does **not** submit forms automatically.
